## 実装状況凡例

| 記号 | 意味 | 説明 |
|------|------|------|
| ✅ | 完全実装 | 機能が完全に実装され、テスト済み |
| 🚧 | 部分実装 | 基本機能は実装済みだが、一部機能が未完成 |
| 📝 | モック実装 | UIは実装済みだが、バックエンド処理が仮実装 |
| ❌ | 未実装 | 機能が未実装 |
